Data Mining

Linear regression analysis, two-class classification analysis, significance test and survival analysis are provided by us.
Two-class classification analysis are provided by us. Two-class classification is processed with a widely used Partial Least Square method. Because data-mining is a highly sophisticated research which is usually need several steps with different algorithms. Therefore, advanced data-mining analysis is available on requirement.

News & Updates

Until now, all data was downloaded from the “Legacy GDC portal” which was updated before 10/05/2016. New updated data is coming!

Gene expression is the level 3 data preprocessed by TCGA was used which was measured by the platform Illumina HiSeq 2000 RNA Sequencing Version 2.

The level 3 CNV (Copy Number Variation) data was measured by Affymetrix Genome-Wide Human SNP Array 6.0

The level 2 Methylation data was collected via the HumanMethylation 450 platform.
